2. Ejemplos de Uso

Usando replace

El div contenedor original desaparece y es reemplazado completamente por el div del fragmento.

(Este texto y este div desaparecerán)

Usando insert

El fragmento se inserta dentro del div contenedor original. El div original se conserva.

(Este texto desaparecerá, pero el div con borde punteado se queda y envuelve al fragmento)

Usando include

El fragmento se incluye dentro dentro del div contenedor original. Pierde las clases/estilos del fragmento.

(Este texto desaparecerá, pero el div con borde punteado se queda y envuelve al fragmento)
